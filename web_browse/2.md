# 2. URLををもとに接続したい相手を探す

- URLを解読した結果のホスト名から接続先を探します。
- DNSにこのホスト名のIPアドレスを教えてください、と問い合わせするとDNSがIPアドレスを教えてくれます。
- 細かいことは補足を読むんだ。

接続先のhost(=ドメイン)がどこにあるのか(IPアドレス)をDNSに問い合わせ、接続先のIPアドレスを取得します
